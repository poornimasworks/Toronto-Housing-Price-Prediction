## Toronto Housing Price Prediction
This Project is to predict the Toronto Housing Prices using machine learning techniques and visualizing market insights through Power BI dashboards
It includes data collection, data cleaning, exploratory data analysis(EDA), Feature Enginnering, splitting the data, training the model and Evaluating the model. 
## Project Overview
The Toroto real estate market is highly dynamic. This project builds a supervised learning model to estimate housing price based on features like location, composite index etc.
The goal is to help the buyers, sellers and real estate analyst make **data-driven decisions**. 
## Objective
- To predict the housing price.
- To identify the factors influencing housing price in Toronto.
- To analyse the neighbourhood benchmark.
